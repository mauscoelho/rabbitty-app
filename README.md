# Rabbitty 🐇

**A keyboard-driven, worklane terminal for macOS that coordinates your AI coding
agents** — see at a glance which agents are working, done, or waiting on you, and
jump between them instantly.

> This is the public home for **downloads, releases, and issues**. The source is
> private; this repo hosts the app and its update feed.

---

## Download

**[⬇︎ Download the latest for Mac](https://github.com/mauscoelho/rabbitty-app/releases/latest)**

1. Download `Rabbitty-Beta-*.dmg` from the latest release.
2. Open it, drag **Rabbitty Beta** onto **Applications**.
3. Launch it from Spotlight.

Signed + notarized by Apple, so it opens with no warning. Universal — works on
**Apple Silicon and Intel**, macOS 13 (Ventura) or newer.

### Or via Homebrew

```bash
brew install --cask mauscoelho/tap/rabbitty-beta
```

Either way, **Rabbitty updates itself** — when a new beta ships, an **Update**
button appears in the top-right; click it to download and relaunch.

---

## What it does

- **Worklanes** — every terminal is a lane; a sidebar shows them grouped by
  workspace with live status.
- **AI-agent status** — Rabbitty reads whether Claude Code, Codex, Antigravity,
  etc. are **working**, **done**, or **blocking on you**, and surfaces it in the
  sidebar, menu bar, and native notifications.
- **Keyboard-first** — command palette (`⌘⇧P`), quick switcher (`⌘P`), everything
  bindable.
- **Native** — real macOS menu bar, notifications, and menu-bar status item.
